## Recomendaciones

- **Formato**: JPG o WebP para mejor compresión
- **Tamaño**: Mínimo 800x600px para buena calidad
- **Aspecto**: Preferiblemente 4:3 para las tarjetas
- **Peso**: Optimizar a menos de 200KB por imagen
